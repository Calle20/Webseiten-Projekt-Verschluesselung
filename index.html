Verschlüsselung von Daten

In unserer heutigen, digitalen Welt wird die Verschlüsselung von Daten immer wichtiger. In den sozialen Medien, beim Online-Shopping oder beim Besuch von Webseiten. Überall geben wir persönliche Daten von uns preis. Doch auch auf unseren PC's oder auf unseren Handys finden sich viele Daten. Wir alle wollen, dass diese Daten geschützt werden und nicht zu betrügerischen Zwecken gestohlen oder gelesen werden. Hier kommt die Verschlüsselung von Daten ins Spiel. Heutzutage gibt es viele verschiedene Möglichkeiten Daten zu codieren. Sie werden ständig weiterentwickelt um den Cyberkrimminellen immer ein Schritt vorraus zu sein. Es sind aber auch viele historische Verschlüsselungsmethoden bekannt. Auf dieser Webseite möchten wir einige historische und aktuelle Methoden zur Verschlüsselung von Daten vorstellen. Einige können sogar auf dieser Seite ausprobiert werden ;-)

Definition Verschlüsselung von Daten

Unter einer Verschlüsselung versteht man die Konvertierung von Daten aus einem lesbaren in ein verschlüsseltes (kodiertes) Format. Um die Daten wieder zu entschlüsseln, braucht man meistens einen Schlüssel, mit dem die Daten verschlüsselt wurden. Jede Verschlüsselung funktioniert mit einem Verschlüsselungsalgorithmus. Mit ihm werden die Daten in einem festgelegten Ablauf ver-/enschlüsselt. Jeder Verschlüsselungsalgorithmus kann in der Theorie durch den Einsatz von Rechenleistung geknackt werden. Je länger ein Computer zum Knacken einer Verschlüsselung braucht, desto sicherer ist sie.
