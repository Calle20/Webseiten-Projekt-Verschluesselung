Verschlüsselung von Daten

In unserer heutigen, digitalen Welt wird die Verschlüsselung von Daten immer wichtiger. In den sozialen Medien, beim Online-Shopping oder beim Besuch von Webseiten. Überall geben wir persönliche Daten von uns preis. Auch auf unseren PC's oder auf unseren Handys finden sich viele Daten. Wir alle wollen, dass diese Daten geschützt werden und nicht zu betrügerischen Zwecken gestohlen oder gelesen werden. Hier kommt die Verschlüsselung von Daten ins Spiel. Früher hauptsächlich von Spionen und dem Militär genutzt, wird die Verschlüsselung heute bei fast jeder Datenübertragung genutzt. Es gibt viele verschiedene moderne Möglichkeiten Daten zu codieren. Sie werden ständig weiterentwickelt um den Cyberkrimminellen immer ein Schritt vorraus zu sein. Doch bereits die Alten Ägypter benutzten Verschlüsselungen, wie beispielsweise das hieroglyphenalphabet. Auf dieser Webseite möchten wir einige historische und aktuelle Methoden zur Verschlüsselung von Daten vorstellen. Einige können sogar ausprobiert werden.

Was ist die Verschlüsselung von Daten

Die Wissenschaft, die sich mit der Verschlüsselung beschäftigt, heißt Kryptographie( griesch. krypto= geheim + grafie= schreiben, Schrift ). Eine Verschlüsselung lässt sich als Verfahren beschreiben, bei dem man einem kryptografischen Algorithmus den Klartext und einen Schlüssel übergibt und einen Geheimtext als Ausgabe erhält. Unter einer Verschlüsselung kann man deshalb die Konvertierung von Daten aus einem lesbaren in ein verschlüsseltes (kodiertes) Format verstehen. Um die Daten wieder zu entschlüsseln, braucht man meistens wieder den Schlüssel, mit dem die Daten verschlüsselt wurden. Jeder Verschlüsselungsalgorithmus kann in der Theorie durch den Einsatz von Rechenleistung geknackt werden. Je länger ein Computer zum Knacken einer Verschlüsselung braucht, desto sicherer ist sie.
